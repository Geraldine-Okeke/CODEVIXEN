USER LOGIN PAGE

Lab Practical - 3

Objective : To create a simple login page using html

Theory : The <form> element is used to create the form, and the action attribute which tells the browser whereto send the form data when the user submits it. the method attribute is set to 'post' which tells the browser to send the data in the body of the HTTP request

The <label> element is used to create a text label for the input field and for attribute is used to associate the label with the corresponding input field.the <input> elements are used to create the text fields for the username and password. the "type" attribute is set to "text" for the username field and "password" for the password field which tells the browser to mask the characters entered in the password field

The required attribute is added to the input, which tells the browser that these fields must be filled out beforethe form must be submitted'

The <input type="submit"> is used to create asubmit button and the value attribute is set to "Login" which will be displayed on the button

There are several other inputs which are radio and checkboxes
